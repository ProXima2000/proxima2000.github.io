♿ Accessibility Statement

I want everyone to feel welcome and comfortable using this website — including people with different access needs.

I try to make sure everything here is clear, easy to read, and works well whether you're on a phone, tablet, computer, or using assistive tech. That means simple language, readable fonts, helpful image descriptions, and easy-to-use navigation.

If you ever have trouble reading something, finding what you need, or using the site in any way — just let me know. I'm happy to help and can send info in a format that works better for you.

Thanks for stopping by!
